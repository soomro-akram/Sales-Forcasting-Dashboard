# Customer-Churn-Prediction

📝 Overview
This Sales Forecasting Dashboard is an interactive web application designed to analyze historical sales data and forecast future sales trends. It leverages machine learning (Prophet / ARIMA) to provide actionable insights, allowing business stakeholders to make data-driven decisions regarding inventory, staffing, and marketing strategies.

The dashboard provides a clear, visual interface for exploring time-series data, understanding seasonal patterns, and viewing confidence intervals for predictions.

🚀 Key Features
📈 Interactive Time-Series Analysis: Visualize historical sales data with zoom and filter capabilities.

🔮 Accurate Forecasting: Utilizes Facebook Prophet (or ARIMA) to generate sales predictions for a customizable period (e.g., 3, 6, or 12 months).

📊 Data Segmentation: Filter data by Product Category, Region, or Store to get specific forecasts.

📉 Performance Metrics: Displays key metrics like Mean Absolute Error (MAE), Root Mean Square Error (RMSE), and Trend direction.

🎨 Dynamic Visualizations: Clean and interactive charts powered by Plotly and Matplotlib.

💾 Data Export: Download forecasted data as CSV files for offline analysis.
